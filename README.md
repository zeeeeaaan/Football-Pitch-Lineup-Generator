# Football-Pitch-Lineup-Generator
Web-based football lineup tool built with HTML, CSS, JS.  Features:  Drag-and-drop players on a half-pitch.  Editable player names and positions.  Color picker for player markers.  Standard formations (4-4-2, 4-3-3, 3-5-2) + custom positions.  Download lineup as JPEG with pitch and player info.
